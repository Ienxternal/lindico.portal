# LindiCo Portal

This folder now contains the standalone client/admin portal app.

To move it into its own repository, copy the contents of `apps/portal` into the
new repo root, install dependencies, add your environment variables, and run:

- `npm install`
- `npm run dev`

Required environment variables:

- `VITE_AUTH0_DOMAIN`
- `VITE_AUTH0_CLIENT_ID`
